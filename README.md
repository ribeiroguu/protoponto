# Protoponto

O Protoponto é um aplicativo mobile desenvolvido em Flutter focado no registro de frequência e controle de ponto de colaboradores em ambiente corporativo. O projeto nasce para substituir um sistema atual ineficiente, marcado por falhas técnicas, dependência de internet, falta de validação lógica e processos manuais passíveis de erros.

A principal premissa do projeto é a autonomia e integridade. Ele opera de forma 100% offline, utilizando o banco de dados local Hive para persistir as informações diretamente no dispositivo do usuário. Isso elimina falhas por falta de conectividade e garante que o estado do registro seja mantido mesmo se o aplicativo for fechado.

A motivação para o desenvolvimento do Protoponto é a necessidade de um sistema de registro de frequência e controle de ponto eficiente, que resolva as dores do funcionários da empresa júnior. Como a inconsistência de dados, a instabilidade técnica e o cálculo ineficiênte.

## Requeriments

- Flutter SDK
- Dart SDK
- Hive
